# HotelApp

- Разработана WPF-система управления отелем на C# с использованием .NET и SQLite.
- Реализованы модули регистрации пользователей и сотрудников.
- Поддерживается управление номерами, включая редактирование данных прямо в таблице.
- Добавлена возможность бронирования и просмотра информации о клиентах и комнатах.
- Все данные хранятся в локальной базе SQLite.
- Поддерживается импорт начальных данных из Excel и ручное добавление записей.
- Интерфейс включает админ-панель с модальными окнами, анимациями и удобной навигацией.
